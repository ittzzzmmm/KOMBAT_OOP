## Setup
1. ### install package

    - First, change directory to :

        `cd src/main/client`

    - Then, bun install

        `bun install`
2. ### Use maven
    - Right click at pom.xml file
    - click " Import as Maven Project "
3. ### Run main application
    - run DemoApplication.java
